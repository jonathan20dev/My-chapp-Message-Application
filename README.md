With our application you have a variety of use cases for chat, including:

***Messages are encrypted!***

- Send text messages
- Send audios
- Send images
- Send videos

## Installation

### Clone

`git clone https://github.com/jonathan20dev/my-chapp.git`

### Install node_modules

`npm i`

### Install node_modules

`npm start`
